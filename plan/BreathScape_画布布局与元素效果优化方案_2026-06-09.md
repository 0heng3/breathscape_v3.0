# BreathScape 画布布局与元素效果优化方案

## 当前问题

1. 画布页右侧反馈卡使用绝对定位叠在 `GardenStage` 上，会遮住右侧可绘制区域。
2. 首页七日按钮偏系统标签，不够像儿童 App 的花园入口。
3. 部分元素虽然已经有独立 `response-*` 类，但同类元素仍然相似，例如光点、星点、植物竖线和水面圆环。
4. 音效需要继续保持 Web Audio API 合成，但同一类元素要有更清晰的音色和节奏差异。

## 布局方案

### 画布页

- 将右侧反馈卡从“画布上绝对定位”改为“画布旁边的同级侧栏”。
- iPad 横屏下采用两列：左侧画布，右侧回应栏，底部工具栏跨两列。
- 当宽度不足时，回应栏降到底部，不遮挡画布。
- 顶部反馈气泡保留，但宽度避开右侧栏。

### 首页七日按钮

- 参考 `plan/fig` 示例：圆润胶囊、柔和渐变、轻微厚度、内发光、小图形标识。
- 每天按钮展示数字、花园名和一个小图形：
  - 种子庭院：种子芽
  - 小溪花园：溪流线
  - 风铃草坡：风铃
  - 石径安放园：石头
  - 雨后蘑菇园：蘑菇和水洼
  - 萤火灯园：小灯和光点
  - 星光温室：星星

## 元素逐个设计

- 种子：小种子落地并轻弹。
- 一点草：细草叶生长，并按风向慢摆。
- 晨光：小圆光斑落下，不等同于长射线。
- 露珠：水珠闪烁并停在草叶附近。
- 土纹：横向纹理铺开。
- 小花：花瓣从中心打开。
- 水线：溪流线横向滑动。
- 涟漪：圆环从中心扩散。
- 叶舟：叶片在水面漂移并轻摆。
- 小桥：木板一段段铺开。
- 雨滴：短竖线下落并留下湿痕。
- 芦苇：高窄草茎带穗，摆动幅度小。
- 风线：透明弧线横向扫过并带动植物。
- 风铃：小铃铛摆动并发清脆短音。
- 彩带：多层彩色弧线飘动。
- 云团：胖云块慢漂，低频柔声。
- 飘叶：叶片旋转漂移。
- 石头：不规则石块落地，低音短促。
- 苔藓：小点群向外蔓延。
- 慢光：呼吸式圆光晕。
- 小树：树干和树冠向上生长。
- 柔影：半透明阴影横向铺开。
- 路牌：竖杆加牌子立起。
- 蘑菇：伞盖从土中弹出。
- 嫩芽：双叶向上展开。
- 水洼：扁椭圆水面亮一下。
- 银线：柔亮轨迹缓慢延伸。
- 花苞：小圆苞轻轻点头。
- 灯笼：暖色灯体和外发光，慢闪。
- 萤火：小圆光点不规则飞行。
- 月亮：月牙升起。
- 窗光：远处窗格亮起。
- 夜花：紫蓝色花瓣慢慢打开。
- 轻风：细弧线慢扫并带动灯光。
- 星星：星形闪烁。
- 记忆种子：双色小种子脉冲。
- 星座线：点与点之间生成细线。
- 月光：斜向柔光束扫过温室。
- 淡彩虹：多色弧线展开并停留更久。

## 实施约束

- 不重构路由和主流程。
- 不接入外部音频，继续使用 Web Audio API 合成。
- 不使用摄像头或后端。
- 文案保持非诊断、非评判。
- 1024×768 横屏优先可用，不遮挡绘画主区域。
